What you are seeing

This simulator opens on the disk structure topic. You can inspect the partition area, filesystem metadata, inode table, directory entries, and raw file blocks.

Quick Notes

MBR: the traditional partition record at the beginning of a disk.

Filesystem metadata: structures such as the superblock, allocation data, and inode tables.

File carving: recovery by scanning raw blocks for signatures like FFD8 and FFD9, even when directory and inode links are broken.

Live forensics: captures volatile evidence before shutdown, especially RAM and network state.

2. File Carving: Scan Raw Blocks for File Signatures

This animation shows a deleted JPEG whose directory entry is gone. The recovery tool ignores the broken filesystem links and scans raw blocks directly. It moves block-by-block until it sees a known header. For JPEG, the common start marker is FFD8 and the footer is FFD9.
